## Getting Started

### Usage
To create the library:
```
make
```

To use in the project:
```
gcc <flags> <sources> -L./path_to_libft.a -lft
```

To clean the library:
```
make clean
make fclean #Delete also libft.a
```

To clean the library and recompile
```
make re
```
